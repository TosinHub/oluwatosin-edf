# Oluwatosin EDF Assessment

## Project Overview

This project is a client for searching books from different sellers based on author, publisher, or year. It demonstrates the implementation of fetching data from various APIs, parsing JSON and XML responses, and handling errors.

## Features

- Fetch books by author
- Fetch books by publisher
- Fetch books by year
- Handle different response formats (JSON and XML)
- Error handling for failed requests

## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/TosinHub/oluwatosin-edf.git
    ```
2. Navigate to the project directory:
    ```bash
    cd oluwatosin-edf
    ```
3. Install dependencies:
    ```bash
    npm install
    ```
